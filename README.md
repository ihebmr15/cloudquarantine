# CloudQuarantine

CloudQuarantine is an event-driven Kubernetes security control plane that detects runtime threats, evaluates them through a risk-based decision engine, and executes automated response playbooks including quarantine, remediation, and forensic logging.

## Core capabilities
- Runtime threat detection with Falco
- Event normalization and routing
- Risk-based decision engine
- Automated quarantine playbooks
- Forensic evidence collection
- Recovery workflows
- Incident metrics and timelines
